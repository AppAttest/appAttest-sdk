# Security Policy

## Reporting a vulnerability

Email **security@appattest.dev** with details. Do not file public issues
for security problems. We aim to acknowledge within one business day.

If you have a fix in mind, attach a private patch — we'll discuss
coordinated disclosure before merging.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
| < 0.1   | No        |

## Scope

This policy covers the AppAttest Swift SDK (this repository) and its
companion `AppAttestObjC` wrapper. Issues with the AppAttest API
service, dashboard, or website should be reported to their respective
contact channels at https://www.appattest.dev/security.
